# Języki Znacznikowe - Archiwum zadań

Katalog zawiera dokumentację oraz komplet zadań laboratoryjnych realizowanych w ramach przedmiotu Języki Znacznikowe. Repozytorium stanowi chronologiczny zapis nauki tworzenia nowoczesnych dokumentów internetowych.

## Zakres materiału

W ramach archiwum znajdują się pliki realizujące następujące zagadnienia:

* **Struktura HTML5**: Budowa dokumentów, semantyka (header, nav, main, section, aside, footer) oraz meta-tagi.
* **Formatowanie treści**: Nagłówki, akapity, listy numerowane i nienumerowane, tabele oraz łączenie komórek.
* **Multimedia**: Osadzanie grafik (`img`), dźwięku (`audio`) oraz wideo (`video`) wraz z atrybutami kontrolnymi.
* **Formularze**: Tworzenie interaktywnych pól wejściowych (text, password, radio, checkbox, select, textarea).
* **Kaskadowe Arkusze Stylów (CSS)**: 
    * Metody dołączania: inline, wewnętrzne oraz zewnętrzne arkusze `.css`.
    * Model pudełkowy (Box Model): marginesy, padding, obramowanie.
    * Pozycjonowanie: praca z `float` oraz budowanie układów kolumnowych.

## Struktura katalogów

* `lab01` – Wprowadzenie do HTML i tabel.
* `lab06` – Nawigacja, ścieżki relatywne i SEO.
* `lab07` – Zaawansowane formularze i zbieranie danych.
* `lab08` – Układy semantyczne i multimedia.
* `lab09` – Kaskada stylów i metody integracji CSS.

## Technologie

* **HTML5** (HyperText Markup Language)
* **CSS3** (Cascading Style Sheets)
