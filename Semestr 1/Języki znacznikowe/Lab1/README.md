# Języki Znacznikowe - Laboratorium 1: Podstawy HTML i Tabel

Repozytorium zawiera pliki z pierwszego laboratorium dotyczącego tworzenia strukturalnych dokumentów tekstowych przy użyciu języka HTML5 oraz ich stylizowania za pomocą kaskadowych arkuszy stylów (CSS).

## Zakres materiału

* **Struktura dokumentu HTML5**: Deklaracja typu dokumentu, sekcja nagłówkowa (`<head>`) oraz ciało dokumentu (`<body>`).
* **Formatowanie tekstu**: Wykorzystanie nagłówków (`<h3>`) oraz znaczników przełamania linii (`<br>`).
* **Tabele danych**: Konstrukcja tabeli przy użyciu znaczników `<table>`, wierszy `<tr>`, nagłówków kolumn `<th>` oraz komórek danych `<td>`.
* **Identyfikatory (ID)**: Przypisywanie unikalnych identyfikatorów do elementów w celu ich późniejszej stylizacji (np. `blok1`, `pozycja1`).
* **Relacje zewnętrzne**: Podpinanie zewnętrznego arkusza stylów CSS za pomocą znacznika `<link>`.

## Struktura projektu

* `index.html` – Główny plik z treścią i strukturą tabeli wyników Mistrzostw Świata w Katarze (Grupa C).
* `style1.css` – Arkusz stylów definiujący wygląd tabeli, kolory wierszy oraz układ kontenera.

## Instrukcja uruchomienia

1. Upewnij się, że pliki `index.html` oraz `style1.css` znajdują się w tym samym folderze.
2. Otwórz plik `index.html` w dowolnej przeglądarce internetowej (Chrome, Firefox, Edge).
