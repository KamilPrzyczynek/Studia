# Movie Browser

Projekt laboratoryjny wykonany w React + TypeScript z wykorzystaniem React Query v5 oraz API TMDB.

## Funkcjonalności

- pobieranie popularnych filmów z TMDB
- wyszukiwanie filmów z debouncingiem
- modal ze szczegółami filmu
- infinite scroll
- obsługa stanów loading, error, empty i success
- ulubione filmy zapisane w localStorage
- mockowanie błędów HTTP przy użyciu MSW
- React Query DevTools

## Technologie

- React 18
- TypeScript
- Vite
- TanStack React Query v5
- Axios
- MSW
